🔹 What each file is and how to explain it (short, writer-friendly)
model_comparison.csv

Contains rows: model name, accuracy, macro_f1.
Explain: “This table compares baseline models (LogisticRegression, MultinomialNB, LinearSVC, RandomForest) using test-set accuracy and macro-F1. Macro-F1 averages F1-scores across classes equally and is preferred for balanced evaluation across emotions.”

LogisticRegression_classification_report.csv

Contains per-class precision, recall, f1-score, support.
Explain:

Precision = when model predicts a class, how often it was correct.

Recall = of all real examples for a class, how many the model found.

F1-score = harmonic mean of precision & recall; best single per-class metric.

Support = number of true examples in test set for that class.

Use these CSV numbers to produce a short table or to quote top lines in Results.

LogisticRegression_confusion_matrix.csv

Square table where rows = true labels and columns = predicted labels.
Explain: Each cell shows how many items of class A were predicted as class B; the diagonal shows correct predictions. Use heatmap to visually present confusion patterns—e.g., “fear often misclassified as anxiety/anger”.

model_accuracy_comparison.png

Bar chart of accuracy for the four baseline models — place this as a figure and refer to it when saying “Logistic Regression achieved highest accuracy”.

🔹 Suggested short paragraph for the thesis (copy-paste)

Baseline model training and evaluation

We trained four baseline classifiers on TF-IDF features — Logistic Regression, Multinomial Naive Bayes, Linear SVC, and Random Forest — using an 80/20 stratified split of the cleaned dataset. Each model was trained on the sparse TF-IDF matrices (unigram+bigram, 5000 features) and evaluated on the held-out test set. Table X summarizes test accuracy and macro-F1 for each model. Logistic Regression achieved the highest accuracy (0.52295) and macro-F1 (0.5192) and was selected as the primary baseline for further tuning. Per-class precision, recall, and F1 metrics are reported in the classification report, and the confusion matrix (Figure Y) shows that fear is the most confused class, frequently predicted as anxiety or anger.
